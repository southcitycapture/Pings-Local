# Pings v2 (Tauri)

Pings v2 is the macOS-first migration from Electron to Tauri, with Linux planned after mac parity.

## Run

```bash
cd /Users/zachjack/Apps/Pings
npm install
npm run dev
```

## Build Targets

```bash
npm run build:mac:arm64
npm run build:mac:x64
# later
npm run build:linux:x64
```
